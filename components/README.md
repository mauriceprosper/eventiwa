# Shared markup snippets

These are plain HTML partials. Since this is a no-build vanilla project, there's
no server-side include — copy the markup from `nav.html` and `footer.html` into
the top and bottom of each page's <body>, or load them with the small fetch
helper in `assets/js/include.js` (see usage note at the bottom of that file).

Keep them identical across pages so the design system stays consistent.
